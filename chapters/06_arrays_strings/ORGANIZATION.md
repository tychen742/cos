# Chapter 06 Organization: Arrays and Strings

## Chapter Goal

Store related values and traverse them with loops.

## Learning Outcomes

- Create, access, update, and loop over a fixed array.
- Compute an element count for a local fixed array.
- Connect `char[]` to introductory C string behavior.

## Sequence

1. Fixed arrays and indexes.
2. Traversal with loops.
3. Size and element count.
4. Character arrays and strings.

## Systems Lab

Inspect fixed array traversal and C string termination so readers connect contiguous storage, bounds, and project input buffers before pointer depth arrives.

## Deliberately Deferred

- dynamic arrays;
- pointer arithmetic depth;
- advanced string-library coverage.
