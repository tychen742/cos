# Chapter 06 Materials: Arrays and Strings

## Current Material

- fixed integer array creation, indexing, update, and traversal.

## Candidate Additions

- array length pattern with `sizeof`;
- character arrays and string terminator vocabulary;
- array bounds mistake discussion.

## Common Mistakes

- indexing past the last element;
- confusing array size in bytes with element count;
- treating a C string as magic rather than a character array convention.
