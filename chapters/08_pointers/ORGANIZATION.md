# Chapter 08 Organization: Pointers

## Chapter Goal

Build the memory vocabulary needed for C data flow and allocation.

## Learning Outcomes

- Explain address-of and dereference operations in small examples.
- Declare and print pointer values correctly.
- Connect pointer parameters and array examples to earlier chapters.

## Sequence

1. Addresses and pointer variables.
2. Dereferencing and mutation through a pointer.
3. Pointer parameters.
4. Arrays and pointers.

## Deliberately Deferred

- dynamic allocation details until Chapter 10;
- advanced pointer arithmetic.
