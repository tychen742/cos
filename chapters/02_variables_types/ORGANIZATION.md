# Chapter 02 Organization: Variables and Types

## Chapter Goal

Give readers the value and type vocabulary needed for expressions and control flow.

## Reader Prerequisites

- first C program shape;
- basic output.

## Learning Outcomes

- Declare and initialize variables.
- Distinguish `int`, `float`, `double`, and `char` in introductory examples.
- Print values with suitable format specifiers.
- Use `sizeof`, casts, and constants in small examples.

## Sequence

1. Values, identifiers, declarations, and assignment.
2. Primitive types and literals.
3. Formatting values with `printf`.
4. Size and conversion.
5. Constants and short review.

## Systems Lab

Compare the sizes and printed forms of a few values so the reader sees that a type controls representation choices, storage size, and how project data can be interpreted.

## Deliberately Deferred

- full string handling;
- arrays as a general collection;
- integer ranges and representation details beyond motivation.
