# Chapter 07 Organization: Functions

## Chapter Goal

Use functions to name operations and manage program size.

## Learning Outcomes

- Declare, define, and call a function.
- Pass values through parameters and use return values.
- Explain why array-processing functions need size information.
- Read a small recursive example.

## Sequence

1. Function shape and calls.
2. Parameters and return values.
3. Scope and decomposition.
4. Arrays with functions.
5. Recursion preview.
